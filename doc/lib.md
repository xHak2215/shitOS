### функции и бибиотеки:
`multi_lib.c` - набор функций раной направлености, заменяет чать страндартных либ C
`text_tool.c` - набор функций для работы с текстом, вывод, декоировка ввода, таблицы VGA цваетов  

`multi_lib.c`:
- `strlen` - считывает синуолы в стоке
- `int_to_str` - числа в стоки
- `strncasecmp` - сравнивает 2 строки без учета регистра
- `chec_symbol` - считвает сколько указаных синволов находиться в строке
- `append` - добовляет одну строку в конец другой

`text_lib.c`:
- `keyboard_map` - массив с синволоми для расшифровки получаемых сигналов для клавиатуры
- `color` - получение VGA кода цвета по названию, 1 аргумент цвет текста 2 аргумент цвет фона 
- `clear_screen` - очистка экрана, аргусетаом передаётся цвет заднего фона(его код)
- `print` - вывод текста на экран, 1 аргумент это выводимый текст, 2 аргумент цветовой код
